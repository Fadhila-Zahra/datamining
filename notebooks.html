Data Nominal#

Data nominal merupakan salah satu data yang bersifat kategorik. Data ini diperoleh dari pengukuran nominal, yaitu proses pengelompokan beberapa objek yang berbeda berdasarkan karakteristik tertentu. Data ini hanya memungkinkan setiap objek memiliki satu kategori

Melakukan pengukuran terhadap data nominal dengan menggukan rumus :
\[ d(i,j)=\frac{p - m}p \]

\( p \) = jumlah data yang akan dihitung jaraknya
\( m \) = jumlah data yang sama

def d1(i,j):
    P = 4
    M = 0
    for col in ["brand_name","model_name","os", "release_date"]:
        if phone[col][i] == phone[col][j]:
            M += 1

    return (P-M)/P

Melakukan perhitungan menggunakan rumus euclidean distance Rumus :

\[ d(i,j) = \sqrt(|{x_{i1} - x_{j1}}|^2 + |{x_{i2} - x_{j2}}|^2 + ... + |{x_{ip} - x_{jp}}|^2) \]
# Menghitung jarak Numeric dengan metode Euclidean
def d2(i,j,h):
    result = 0
    for col in [hasil1, hasil2, hasil3, hasil4, hasil5, hasil6, hasil7, hasil8]:
        result = math.pow(abs(col[i] - col[j]), h)
    return result**(1/h)

# Hasil perhitungan

\( k \) = jumlah penglompokan

Tugas 5 (K-Means Clustering)#

Clustering adalah proses untuk mengelompokkan data dalam beberapa cluster atau kelompok sehingga data dalam satu cluster memiliki tingkat kemiripan yang maksimum dan data antar cluster yang berbeda memiliki kemiripan minimum. Clustering untuk mengelompokkan data berdasarkan klasifikasi yang sama.

K-means mempunyai kemampuan mengelompokkan data dalam jumlah yang cukup besar dengan waktu komputasi yang cepat dan efisien. K-Means merupakan salah satu algoritma klastering dengan metode partisi (partitioning method) yang berbasis titik pusat (centroid) selain algoritma k-Medoids yang berbasis obyek.

Algoritma k-means yaitu :

  1. Langkah 1, menentukan secara acak K titik data sebagai pusat cluster yang disebut centroid.

  2. Langkah 2, menandai masing-masing xi masuk ke cluster tertentu, dengan cara menghitung jarak xi ke masing-masing pusat cluster(centroid) dan memasukkan xi anggota pusat cluster tertentu tersebut jika memiliki jarak terdekat.

  3. Langkah 3, menentukan pusat cluster baru dengan menghitung rata-rata dari anggota cluster.

  4. Langkah 4, ulangan
